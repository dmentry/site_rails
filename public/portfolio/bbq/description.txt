Web-приложение для организации мероприятий, подписки на них, комментирования, получений уведомлений и загрузки фотографий.<br>
Основной функционал:
<ul>
<li>&bull;&nbsp;создание мероприятия с привязкой к карте</li>
<li>&bull;&nbsp;подписка на мероприятие для отслеживания изменений - уведомления по email</li>
<li>&bull;&nbsp;возможность добавлять комментарии и фотографии</li>
<li>&bull;&nbsp;возможность создавать "секретные" мероприятия (доступ по пин-коду)</li>
<li>&bull;&nbsp;доступ и для незарегестрированных пользователей</li>
</ul>
