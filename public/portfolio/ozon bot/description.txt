Бот для помощи просмотра и редактировния списка товаров, находящихся в акциях. А также сигнализирования и\или удаления товара из акции, если он не из списка исключений и добавляется в акции Озоном автоматически.<br>
Основной функционал:
<ul>
<li>&bull;&nbsp;<strong>Показать все товары в акциях</strong> - показывает товары, находящиеся в акциях, в любых статусах, кроме архивных. При нажатии на конкретный товар выводится меню с командами. Для товаров в активном статусе для каждой акции доступно ее добавление в список исключений и удаление товара из этой акции. Для товаров в неактивных статусах акции невозможно добавить в список исключений. Можно только удалить такие товары из акций, в которых они находятся. Если удаленная акция для конкретного товара есть в его списке исключений, она удаляется оттуда автоматически</li>
<li>&bull;&nbsp;<strong>Показать список исключений</strong> - список, в который вносятся конкретные акции для каждого выбранного товара. Это сделано для гибкой настройки нахождения товаров в акциях: бот не удаляет товары из акций, находящихся в этом списке. Товары удаляются из всех акций, которых нет в списке исключений. Т.е., те, в которые добавит конкретный товар Озон без вашего уведомления</li>
<li>&bull;&nbsp;<strong>Добавить товары в список исключений</strong> - показывается список всех товаров, за исключением архивных. При нажатии на конкретный товар выводятся все доступные на данный момент акции. Каждую из них можно добавить в исключения для данного товара</li>
<li>&bull;&nbsp;<strong>Автоудаление товаров из акций (вкл\выкл)</strong> - включает\выключает автоматическое удаления товаров, добавленных Озоном в акции, которые не находятся в списке исключений. Если эта опция выключена, то бот просто информирует о добавлении товаров в акции, но не удаляет из них товары</li>
<li>&bull;&nbsp;<strong>Товары с фотографиями (вкл\выкл)</strong> - включает\выключает отображение главной фотографии при выводе товара в разделах меню</li>
<li>&bull;&nbsp;<strong>Отправка сообщений о неудачных проверках (вкл\выкл)</strong> - включает\выключает отправку сообщений о неудачных проверках (когда сервер Озона не отвечает) при проверке товаров в акциях в автоматическом режиме</li>
</ul>
